# LuSplit Logo Usage

Version 1.0 - Foundational Identity

## Primary Icon Construction

The icon is a rounded square container with a vertical split:

- Left panel: teal
- Right panel: indigo
- Center divider: white
- Divider top detail: small upward-pointing zip-like triangle

Production icon uses flat colors only. No gradients.

### Geometry

- Triangle height: `18–22%` of container height
- Triangle corners: slightly rounded
- Triangle width: `2.5x` divider thickness
- Triangle placement: optically centered on divider
- Divider thickness: `6–8%` of container width
- Container corner radius: `28–32%` of square size

## Logo Variants

### 1) Primary (Light Background)

- Icon above wordmark
- Centered alignment

### 2) Dark Mode

- Background: deep navy (`#0F172A`)
- Icon colors: slightly desaturated
- Divider: white

### 3) Monochrome

- Navy on white
- White on navy

## Wordmark

- Style: rounded sans-serif
- Recommended fonts:
  - `Inter Rounded`
  - `SF Pro Rounded`
  - `Nunito`
  - `Manrope` (fallback)
- Weight: Medium
- Tracking: `+2%`
- Color: navy (`#1F2937`)

Wordmark details:

- Dot of `i` must be perfectly circular
- Keep consistent cap-height alignment with icon

## Usage Rules

- Teal is always left, indigo is always right
- Divider is always white
- No drop shadows in logo files
- No gradients in app store production icon

## App Store Presentation

- Background: snow (`#F6F8FB`)
- Icon centered
- No gradients
- No mock device frames

Screenshot direction:

- Show clarity
- Include “All settled” state where possible

Headline example: “Split fairly. Stay friends.”

# Soporte

LuSplit es un proyecto de código abierto. El soporte se gestiona a través del repositorio público.

## Documentación

El proyecto incluye documentación sobre la arquitectura de la app, el formato de exportación, el modelo de dominio y las pautas de marca. Todo está disponible en el repositorio en la carpeta `docs/`.

## Reportar un error

Si encuentras un error, abre un issue en el repositorio de GitHub. Por favor, incluye:

- Qué estabas haciendo cuando ocurrió el problema
- Qué esperabas que pasara
- Qué pasó realmente
- Tu dispositivo y sistema operativo, si procede

## Solicitudes de funciones

Las solicitudes de funciones pueden abrirse como issues de GitHub. El proyecto está actualmente en su primera versión, centrada en un núcleo mínimo y estable. No todas las solicitudes serán aceptadas, pero se leen todas.

## Contribuir

Las contribuciones son bienvenidas. Consulta el archivo `CONTRIBUTING.md` en el repositorio para ver las pautas sobre cómo contribuir código, documentación o traducciones.

## Contacto

El proyecto no tiene un correo de soporte dedicado. Toda la comunicación se hace a través del repositorio público. Esto mantiene el proyecto transparente y accesible para la comunidad.

## Acerca de

LuSplit fue creado como una alternativa de código abierto a las apps de división de gastos con modelos de pago. Es gratuito y lo seguirá siendo. Las funciones principales nunca estarán bloqueadas tras una suscripción.

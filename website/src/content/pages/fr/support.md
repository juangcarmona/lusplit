# Support

LuSplit est un projet open-source. Le support passe par le dépôt public.

## Documentation

Le projet comprend une documentation couvrant l'architecture de l'application, le format d'export, le modèle de domaine et les directives de marque. Tout est disponible dans le dépôt sous le dossier `docs/`.

## Signaler un bug

Si vous trouvez un bug, ouvrez une issue dans le dépôt GitHub. Veuillez inclure :

- Ce que vous faisiez quand le problème s'est produit
- Ce à quoi vous vous attendiez
- Ce qui s'est réellement passé
- Votre appareil et votre système d'exploitation, si pertinent

## Demandes de fonctionnalités

Les demandes de fonctionnalités peuvent être soumises comme des issues GitHub. Le projet est actuellement dans sa première version, axé sur un noyau minimal et stable. Toutes les demandes ne seront pas acceptées, mais elles sont toutes lues.

## Contribuer

Les contributions sont les bienvenues. Consultez le fichier `CONTRIBUTING.md` dans le dépôt pour les directives sur la façon de contribuer du code, de la documentation ou des traductions.

## Contact

Le projet n'a pas d'adresse e-mail de support dédiée. Toute communication se fait via le dépôt public. Cela maintient le projet transparent et accessible à la communauté.

## À propos

LuSplit a été créé comme une alternative open-source aux applications de partage des dépenses avec des modèles payants. Il est gratuit et le restera. Les fonctionnalités principales ne seront jamais bloquées derrière un abonnement.

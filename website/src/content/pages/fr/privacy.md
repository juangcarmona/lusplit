# Confidentialité

LuSplit est conçu avec une approche simple de la confidentialité : vos données restent sur votre appareil.

## Stockage des données

Toutes les données de dépenses, les groupes et les soldes sont stockés localement sur l'appareil sur lequel vous utilisez LuSplit. Il n'y a pas de backend cloud obligatoire. Aucune donnée n'est envoyée à des serveurs externes lors d'une utilisation normale.

## Pas de comptes

LuSplit ne vous demande pas de créer un compte. Il n'y a pas de formulaire d'inscription, pas d'adresse e-mail et pas de mot de passe. L'application fonctionne immédiatement sans aucune connexion.

## Pas de suivi

LuSplit n'inclut pas de scripts publicitaires, de suivi intersites ou d'analyses tierces. Le site web public n'utilise aucun suivi côté client par défaut.

## Pas de télémétrie

L'application n'envoie pas automatiquement de données d'utilisation, de rapports de plantage ou d'informations de diagnostic à un quelconque serveur.

## Exportez à tout moment

Vous pouvez exporter toutes vos données à tout moment en JSON, CSV ou PDF. C'est ainsi que LuSplit garantit que vous n'êtes jamais bloqué hors de vos propres enregistrements.

## Open source

LuSplit est open-source. Le code source complet est disponible pour inspection. Vous pouvez vérifier ce que l'application fait et ne fait pas.

## Synchronisation future (optionnelle)

Une future version de LuSplit pourra proposer une synchronisation cloud optionnelle pour une utilisation multi-appareils. Si elle est introduite, elle sera optionnelle, clairement expliquée et n'affectera pas les utilisateurs qui préfèrent rester en local.

## Contact

Si vous avez une préoccupation spécifique en matière de confidentialité, vous pouvez contacter le projet via le dépôt public.

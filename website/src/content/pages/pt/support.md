# Suporte

O LuSplit é um projeto open-source. O suporte é prestado através do repositório público.

## Documentação

O projeto inclui documentação sobre a arquitetura da app, o formato de exportação, o modelo de domínio e as diretrizes de marca, disponível no repositório na pasta `docs/`.

## Reportar um bug

Se encontrar um bug, abra um issue no repositório do GitHub. Por favor, inclua:

- O que estava a fazer quando o problema ocorreu
- O que esperava que acontecesse
- O que aconteceu de facto
- O seu dispositivo e sistema operativo, se relevante

## Pedidos de funcionalidades

Os pedidos de funcionalidades podem ser abertos como issues no GitHub. O projeto está atualmente na sua primeira versão, focado num núcleo mínimo e estável. Nem todos os pedidos serão aceites, mas são todos lidos.

## Contribuir

As contribuições são bem-vindas. Consulte o ficheiro `CONTRIBUTING.md` no repositório para obter diretrizes sobre como contribuir com código, documentação ou traduções.

## Contacto

O projeto não tem um endereço de e-mail de suporte dedicado. Toda a comunicação acontece através do repositório público. Isto mantém o projeto transparente e acessível à comunidade.

## Sobre

O LuSplit foi criado como uma alternativa open-source a apps de divisão de despesas com modelos pagos. É gratuito e assim permanecerá. As funcionalidades principais nunca serão bloqueadas atrás de uma subscrição.

# Funcionalidades

O LuSplit fornece as ferramentas de que um grupo precisa para acompanhar os custos partilhados e chegar a uma liquidação justa — nada mais.

## Grupos

Crie um grupo para qualquer situação partilhada: uma viagem, uma casa, um evento familiar. Cada grupo tem a sua própria lista de despesas e o seu próprio conjunto de pessoas. Os grupos são independentes entre si.

## Pessoas

Adicione pessoas a um grupo. Cada pessoa num grupo é um membro. Pode adicionar membros dependentes — por exemplo, crianças ligadas a um adulto — para que os custos possam ser distribuídos com precisão.

## Despesas

Adicione despesas à medida que acontecem. Cada despesa regista:

- O que foi pago
- Quanto custou
- Quem pagou
- Quem está incluído

A divisão pode ser configurada por despesa: de forma igualitária entre todos os membros, por valores exatos, por percentagem ou por peso.

## Saldos

O LuSplit calcula os saldos continuamente. A qualquer momento pode ver, para cada pessoa do grupo, exatamente quanto deve ou lhe é devido. Os saldos são claros e fáceis de ler.

## Plano de liquidação

Quando o grupo estiver pronto para liquidar, o LuSplit gera uma lista mínima de transferências. Este plano reduz o número de pagamentos necessários para levar todos os saldos a zero. O plano é uma sugestão, não uma transação — nenhum dinheiro passa pelo LuSplit.

## Exportar

Pode exportar o registo completo de despesas e saldos a qualquer momento. Os formatos disponíveis incluem JSON, CSV e PDF. Os seus dados pertencem-lhe.

## Offline primeiro

O LuSplit funciona sem ligação à internet. Os dados são armazenados localmente no seu dispositivo. Não existe sincronização na nuvem obrigatória. Opcionalmente, uma versão futura oferecerá sincronização para quem a queira.

## Sem contas necessárias

Não precisa de criar uma conta para usar o LuSplit. Não há formulário de registo, endereço de e-mail nem senha. A app funciona imediatamente sem qualquer tipo de autenticação.

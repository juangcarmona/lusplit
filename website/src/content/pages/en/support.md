# Support

LuSplit is an open-source project. Support happens through the public repository.

## Documentation

The project includes documentation covering the app architecture, export format, domain model, and branding guidelines. These are available in the repository under the `docs/` folder.

## Reporting a bug

If you find a bug, open an issue in the GitHub repository. Please include:

- What you were doing when the issue occurred
- What you expected to happen
- What actually happened
- Your device and operating system, if relevant

## Feature requests

Feature requests can be opened as GitHub issues. The project is currently in its first version, focused on a minimal and stable core. Not all requests will be accepted, but they are read.

## Contributing

Contributions are welcome. See the `CONTRIBUTING.md` file in the repository for guidelines on how to contribute code, documentation, or translations.

## Contact

The project does not have a dedicated support email. All communication happens through the public repository. This keeps the project transparent and community-accessible.

## About

LuSplit was created as an open-source alternative to paywalled expense-splitting apps. It is free and will remain free. The core features will never be locked behind a subscription.

# Datenschutz

LuSplit ist mit einem unkomplizierten Ansatz zum Datenschutz konzipiert: Ihre Daten bleiben auf Ihrem Gerät.

## Datenspeicherung

Alle Ausgabendaten, Gruppen und Salden werden lokal auf dem Gerät gespeichert, auf dem Sie LuSplit verwenden. Es gibt kein obligatorisches Cloud-Backend. Während der normalen Nutzung werden keine Daten an externe Server gesendet.

## Keine Konten

LuSplit erfordert nicht, dass Sie ein Konto erstellen. Es gibt kein Registrierungsformular, keine E-Mail-Adresse und kein Passwort. Die App funktioniert sofort ohne jegliche Anmeldung.

## Kein Tracking

LuSplit enthält keine Werbescripts, kein seitenübergreifendes Tracking und keine Drittanbieter-Analysen. Die öffentliche Website verwendet standardmäßig kein clientseitiges Tracking.

## Keine Telemetrie

Die App sendet keine Nutzungsdaten, Absturzberichte oder Diagnoseinformationen automatisch an irgendwelche Server.

## Jederzeit exportieren

Sie können alle Ihre Daten jederzeit als JSON, CSV oder PDF exportieren. So stellt LuSplit sicher, dass Sie nie vom Zugang zu Ihren eigenen Aufzeichnungen ausgesperrt werden.

## Open Source

LuSplit ist open-source. Der vollständige Quellcode steht zur Überprüfung bereit. Sie können nachvollziehen, was die App tut und nicht tut.

## Zukünftige Synchronisierung (optional)

Eine zukünftige Version von LuSplit kann eine optionale Cloud-Synchronisierung für die Nutzung auf mehreren Geräten anbieten. Falls sie eingeführt wird, ist sie opt-in, wird klar erklärt und beeinträchtigt keine Nutzer, die lokal bleiben möchten.

## Kontakt

Wenn Sie ein spezifisches Datenschutzanliegen haben, können Sie das Projekt über das öffentliche Repository kontaktieren.

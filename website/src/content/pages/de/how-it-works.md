# So funktioniert es

LuSplit folgt einem einfachen Ablauf von Anfang bis Ende.

## 1. Erstellen Sie eine Gruppe

Beginnen Sie mit der Erstellung einer Gruppe. Geben Sie ihr einen Namen, der zur Gelegenheit passt — "Portugal-Reise", "WG-Ausgaben", "Familienwochenende". Eine Gruppe ist ein Container für eine Reihe von Personen und ihre gemeinsamen Ausgaben.

## 2. Fügen Sie Personen hinzu

Fügen Sie alle Personen hinzu, die an der Kostenteilung beteiligt sind. Jede Person wird durch einen Namen identifiziert. Sie können abhängige Mitglieder (wie Kinder) einem Erwachsenen in der Gruppe zuordnen, damit deren Kosten korrekt zugewiesen werden.

## 3. Fügen Sie Ausgaben hinzu

Wenn jemand für etwas Gemeinsames bezahlt, fügen Sie es als Ausgabe hinzu. Erfassen Sie, was bezahlt wurde, wie viel es gekostet hat, wer bezahlt hat und wen es betrifft. Wählen Sie, wie es aufgeteilt werden soll: gleichmäßig, nach genauen Beträgen, nach Prozentsatz oder nach Gewichtung.

## 4. Salden einsehen

LuSplit berechnet Salden automatisch. Jederzeit können Sie die Gruppe öffnen und genau sehen, wer was schuldet. Positive Beträge bedeuten, dass jemandem Geld geschuldet wird. Negative Beträge bedeuten, dass jemand der Gruppe Geld schuldet.

## 5. Abrechnen

Wenn die Gruppe bereit ist, abzurechnen, erstellt LuSplit einen Abrechnungsplan. Das ist eine minimale Liste von Zahlungen, die alle Salden auf null bringt. Zum Beispiel: "Alex zahlt Jordan 20 €. Jordan zahlt Sam 8 €."

Kein Geld fließt über LuSplit. Die tatsächlichen Zahlungen erfolgen direkt zwischen den Gruppenmitgliedern.

## 6. Exportieren

Exportieren Sie jederzeit die vollständige Ausgabenaufzeichnung und Abrechnungsübersicht als JSON, CSV oder PDF. Behalten Sie eine Aufzeichnung für sich oder teilen Sie sie mit der Gruppe.

---

Das ist der vollständige Ablauf. LuSplit ist bewusst einfach gehalten. Es macht eine Sache: Ihrer Gruppe Klarheit über gemeinsame Kosten zu verschaffen.

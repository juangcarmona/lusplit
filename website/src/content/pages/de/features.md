# Funktionen

LuSplit stellt die Werkzeuge bereit, die eine Gruppe benötigt, um gemeinsame Kosten zu verfolgen und zu einer fairen Abrechnung zu gelangen — nicht mehr.

## Gruppen

Erstellen Sie eine Gruppe für jede gemeinsame Situation: eine Reise, einen Haushalt, ein Familienevent. Jede Gruppe hat ihre eigene Ausgabenliste und ihre eigenen Mitglieder. Gruppen sind voneinander unabhängig.

## Personen

Fügen Sie Personen zu einer Gruppe hinzu. Jede Person in einer Gruppe ist ein Mitglied. Sie können abhängige Mitglieder hinzufügen — zum Beispiel Kinder, die einem Erwachsenen zugeordnet sind — damit Kosten genau verteilt werden können.

## Ausgaben

Fügen Sie Ausgaben hinzu, sobald sie entstehen. Jede Ausgabe erfasst:

- Was bezahlt wurde
- Wie viel es gekostet hat
- Wer bezahlt hat
- Wen es betrifft

Die Aufteilung kann pro Ausgabe konfiguriert werden: gleichmäßig unter allen Mitgliedern, nach genauen Beträgen, nach Prozentsatz oder nach Gewichtung.

## Salden

LuSplit berechnet Salden fortlaufend. Jederzeit können Sie für jede Person in der Gruppe genau sehen, wie viel sie schuldet oder ihr geschuldet wird. Salden sind klar und leicht lesbar.

## Abrechnungsplan

Wenn die Gruppe bereit zur Abrechnung ist, erstellt LuSplit eine minimale Liste von Überweisungen. Dieser Plan reduziert die Anzahl der notwendigen Zahlungen, um alle Salden auf null zu bringen. Der Plan ist ein Vorschlag, keine Transaktion — kein Geld fließt über LuSplit.

## Export

Sie können die vollständige Ausgabenaufzeichnung und Salden jederzeit exportieren. Unterstützte Formate umfassen JSON, CSV und PDF. Ihre Daten gehören Ihnen.

## Offline-first

LuSplit funktioniert ohne Internetverbindung. Daten werden lokal auf Ihrem Gerät gespeichert. Es gibt keine obligatorische Cloud-Synchronisierung. Optional wird eine zukünftige Version eine Synchronisierung für diejenigen anbieten, die sie wünschen.

## Kein Konto erforderlich

Sie müssen kein Konto erstellen, um LuSplit zu verwenden. Es gibt kein Registrierungsformular, keine E-Mail-Adresse und kein Passwort. Die App funktioniert sofort ohne jegliche Anmeldung.

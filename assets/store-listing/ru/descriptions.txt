SHORT DESCRIPTION

Отслеживайте общие расходы, узнайте кто сколько должен. Без регистрации.


FULL DESCRIPTION

Общие расходы, чёткие счета.

LuSplit помогает отслеживать расходы любой группы — путешествия, жильё, мероприятия и всё остальное. Записывайте каждый совместный платёж, видите долги с первого взгляда и находите быстрый способ рассчитаться. Без регистрации. Работает полностью офлайн.

• Группы — Создавайте группы для путешествий, жилья или любых совместных расходов. Каждая группа независимо ведёт свои расходы.

• Гибкое разделение — Делите поровну, по сумме, проценту или пользовательскому весу. Предпросмотр влияния на балансы перед сохранением.

• Чёткие балансы — Актуальный баланс каждого участника, всегда обновлён. Никаких вычислений в уме.

• Простой расчёт — LuSplit вычисляет минимальное количество переводов, чтобы свести все балансы к нулю. Деньги не проходят через приложение — это лишь подсказка для действий.

• Полная история расходов — Все расходы в одном месте. Редактируйте или удаляйте старые записи — балансы пересчитываются мгновенно.

• Экспорт всего — Экспортируйте данные в JSON, CSV или PDF в любое время.

• Без аккаунта — Без регистрации. Без входа. Полностью офлайн.

LuSplit не управляет деньгами. Все расчёты происходят напрямую между людьми. Приложение просто делает совместные расходы понятными.
